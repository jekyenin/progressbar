progressBar

Плагин для самой простой реализации загрузочной полоски. Она появляется сверху, двигается и исчезает. Вся такая самостоятельная и вообще..

Первые шаги

Не стоит забывать про подключение скрипта и стиля. Подсветка кода, которую я подключил работает только для javascript, поэтому я просто напишу названия файлов.

progressBar.js progressBar.default.css

Для начала использования плагина необходимо инициализировать

var progress = new ProgressBar();

Для того чтобы отобразить полосу необходимо вызвать метод show

progress.show();

Чтобы задать значение полосе нужно передать значение в метод value

progress.value(50);

Стилизация

Описание классов. Цвет можно указать в скрипте, а все остальное все равно менять придется через основной файл (progressBar.default.css), поэтому вот за что отвечает каждый класс

.js_ProgressBar__parent{} //Родительский объект полосы (задается в скрипте options.place) .js_ProgressBar__outer{} //Контейнер для полосы .js_ProgressBar__outer.js_ProgressBar__show{} //Состояние контейнера, когда полоса на виду .js_ProgressBar__bart{} //Сама полоса

API

Создание экземпляра плагина. Нужно назначить экзмепляр переменной, для дальнейшей работе с ним

var progress = new ProgressBar({ place: "body", });

Отображение полосы. Для созданного экземпляра при необходимости можно вызвать метод show, который создает DOM на странице и отображает элемент

progress.show();

Скрытие полосы. Для созданного экземпляра при необходимости можно вызвать метод hide, который скрывает элемент и удаляет DOM со страницы

progress.hide();

Установка значения. Принимает значения от 0 до 100 и устанавливает ширину полосы в процентах

progress.value(50);

События

События устанавливаются при создании экземпляра через методы

Метод onCreate срабатывает при инициализации плагина

var options = { onCreate : function(){} }

Метод onShow срабатывает при показе полосы

var options = { onShow : function(){} }

Метод onHide срабатывает при скрытии полосы

var options = { onHide : function(){} }

Метод onChange срабатывает при изменении значения

var options = { onChange : function(){} }

Метод onComplete срабатывает при достижении 100%

var options = { onComplete : function(){} }

Метод onEmpty срабатывает при достижении 0%

var options = { onEmpty : function(){} }
